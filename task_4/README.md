# CODSOFT TASK 4
# Spam SMS Detection

This project is developed as part of CodSoft Machine Learning Internship.

## Objective
To build an AI model that classifies SMS messages as Spam or Normal.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Naive Bayes Algorithm

## Steps
1. Loaded SMS dataset
2. Preprocessed the data
3. Converted text into numerical format using TF-IDF
4. Trained Naive Bayes ML model
5. Tested accuracy

## Result
Model achieved 96.68% accuracy.

## Output
Successfully predicts whether an SMS is spam or normal.
