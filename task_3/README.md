# CODSOFT TASK 3
# Customer Churn Prediction

This project is completed as part of CodSoft Machine Learning Internship.

## Objective

To develop a Machine Learning model that predicts whether a customer will leave a service or continue.

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Random Forest Classifier

## Dataset

Customer Churn Modelling Dataset containing:

- Credit Score
- Geography
- Gender
- Age
- Balance
- Salary
- Customer Activity Details

## Working Steps

1. Loaded customer dataset
2. Removed unnecessary columns
3. Converted categorical data into numerical data
4. Split dataset into training and testing
5. Trained Random Forest model
6. Predicted customer churn

## Result

Accuracy Achieved: 86.65%

The model predicts whether:
- Customer Will Leave
- Customer Will Stay
